# Louis Wang — Portfolio Website

Personal portfolio website for Louis Wang, Cyber Security & Digital Forensics graduate based in Perth, Western Australia.

🌐 **Live site:** _add your Cloudflare Pages URL here once deployed_

## About

A fully responsive, single-page portfolio built with vanilla HTML, CSS, and JavaScript. Designed with a cyber security aesthetic — dark theme, particle network background, and smooth animations.

## Features

- Animated particle network canvas background
- Typing effect hero section
- Scroll-reveal animations throughout
- Visitor questionnaire modal (Formspree integration ready)
- Fully responsive — mobile hamburger menu included
- SEO meta tags + Open Graph for LinkedIn/social sharing
- Optimised for performance: rAF-throttled scroll, debounced resize, batched canvas draw calls

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- [Google Fonts](https://fonts.google.com/) — Orbitron, Inter, Share Tech Mono
- [Font Awesome 6](https://fontawesome.com/) — icons
- [Formspree](https://formspree.io/) — visitor form submissions (optional)
- Deployed via [Cloudflare Pages](https://pages.cloudflare.com/)
